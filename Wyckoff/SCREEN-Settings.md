# Screen: Settings

**Route:** `/settings`
**Purpose:** Configure the system without editing YAML files directly

## Features

- Add / remove / deactivate tickers in universe (persists to universe.yaml)
- Adjust composite sell/buy score weights
- Set alert thresholds per alert type
- Trigger manual data download (backfill or incremental)
- View download log: last success, last failure, row count per ticker
- Data health summary: gaps, stale tickers, null counts

# Wyckoff Accumulator

Personal portfolio intelligence system for detecting institutional money flows across asset classes using Wyckoff volume-price analysis.

## Intent

Detects sector rotation and institutional accumulation/distribution patterns across ~40 asset class buckets (equity sectors, bonds, commodities, forex, crypto) using end-of-day price/volume data. Pairs quantitative signals with the operator's macro judgment to generate rebalancing guidance — with deliberate emphasis on sell signals, which are behaviorally harder to act on. Runs locally on free data (yfinance + FRED). No auto-trading. Operator decides.

Workflow

#CommandPurpose
1bin/create_spec.sh <Project> “description”Scaffold spec directory from templates
2bin/validate.sh <Project>Check required files, naming, fields
—(edit spec files)Write concise specs: INTENT, ARCHITECTURE, SCREEN-*, FEATURE-*
4bin/convert.sh <Project> > prompt.mdGenerate AI expansion prompt — feed to AI agent
5bin/build.sh <Project> > prompt.mdTag commit, generate build prompt — feed to AI agent

Each build.sh run creates a git tag: build/PROJECT/YYYY-MM-DD.N
Diff between builds: git diff build/GAME/2026-03-19.1..build/GAME/2026-03-20.1 -- GAME/

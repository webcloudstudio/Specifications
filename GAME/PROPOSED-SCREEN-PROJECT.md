# Screen: Configuration

This file allows editing of the metadata stored for each project 

It should be done by source. 

For example many project specificaitons are in Metadata.md - that should be its own section
Sort based on keys first then priority 

Read Only information such as the information from scripts should be separated by general source

This screen is for a single project

You should be able to edit the values

## Persistence

All data edited on this screen is saved in the database and in the project metadata.md
Persistence is immediate after a field is tabbed out of or the tab itself is exited
